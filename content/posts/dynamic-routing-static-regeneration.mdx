MDX의 고급 서식 알아보기

## 펜싱된 코드 블록
코드 블록 앞뒤에 삼중 백틱 ```을 배치하여 펜싱된 코드 블록을 만들 수 있습니다.

```
function test() {
  console.log("notice the blank line before this function?");
}
```

### 구문 강조
선택적 언어 식별자를 추가하여 펜싱된 코드 블록에서 구문을 강조하도록 설정할 수 있습니다.

구문에서 Ruby 코드를 강조 표시하려면 다음을 수행합니다.

```ruby
require 'redcarpet'
markdown = Redcarpet.new("Hello World!")
puts markdown.to_html
```

## JSX 사용하기
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs
  defaultValue="apple"
  values={[
    {label: 'Apple', value: 'apple'},
    {label: 'Orange', value: 'orange'},
    {label: 'Banana', value: 'banana'},
  ]}>
  <TabItem value="apple">이것은 사과입니다 🍎</TabItem>
  <TabItem value="orange">이것은 오렌지입니다 🍊</TabItem>
  <TabItem value="banana">이것은 바나나입니다 🍌</TabItem>
</Tabs>;
